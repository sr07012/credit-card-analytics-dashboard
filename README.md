# credit-card-analytics-dashboard
SQL and Power BI-based Credit Card Analytics Dashboard for analyzing transaction patterns, customer behavior, revenue trends, and financial KPIs.
📌 Project Overview

This project is a Credit Card Analytics Dashboard developed using SQL for data extraction and Power BI for data visualization.

The dashboard analyzes credit card transactions, customer spending behavior, revenue trends, and key financial performance indicators to provide actionable business insights.

🎯 Objective

To design a data-driven financial reporting system that:

Analyzes credit card transaction data

Identifies revenue patterns

Tracks customer spending behavior

Monitors financial KPIs

Supports data-driven decision-making

🛠 Tools & Technologies Used

SQL (Data Extraction & Querying)

Power BI (Data Modeling & Visualization)

DAX (KPI Calculations)

Data Cleaning & Transformation

🗄 Data Processing Workflow

Extracted transaction data using SQL queries.

Performed data filtering and aggregation.

Imported cleaned dataset into Power BI.

Built data model and relationships.

Created DAX measures for KPIs.

Designed interactive dashboard.

📊 Key Dashboard Features

Total Transaction Volume

Total Revenue

Customer Segmentation

Spending Category Analysis

Monthly Revenue Trends

Average Transaction Value

Profitability Insights
